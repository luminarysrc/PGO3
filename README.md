# Simple Library

A small Java repository representing a basic library borrowing system, built iteratively.

## Classes
- **`Book`**: Stores title, author, total pages, availability, and a counter tracking how many times it was borrowed.
- **`Reader`**: Represents a user borrowing books, referencing their card number and total concurrently borrowed books.
- **`Library`**: Stores a collection of books (using an array) with core operations to add, view, find by title, find by author, and handle borrowing logic.
- **`Main`**: A small scenario illustrating how objects interact together.
